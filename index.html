SALIENT GNN

A software system for accelerating the training and inference of graph neural networks.

Learn more

What is SALIENT?

The SALIENT project focuses on the development of systems for training graph neural networks (GNNs) that are fast, scalable, and accurate. The system, called SALIENT, is designed to operate within existing abstractions in popular libraries for training GNNs, such as PyTorch Geometric, and do not require machine learning practitioners to rewrite their models or adopt a new framework.

The Systems

The SALIENT project has resulted in the development of two systems for training GNNs: SALIENT and SALIENT++. These systems are fast, scalable, and accurate, and operate within existing abstractions in popular libraries for training GNNs. Learn more about the systems and how to use them on this page. The code for SALIENT and SALIENT++ is available on github Github
